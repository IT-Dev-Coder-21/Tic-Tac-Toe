# React Tic-Tac-Toe Game

This is a simple Tic-Tac-Toe game built using React function components and hooks.

## Features
- 2-player game (X and O)
- Detects winner automatically
- Shows move history and allows undoing moves
- Simple, responsive UI

- Uses `App.js`, `Board.js`, and `Square.js` components
